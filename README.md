http://localhost:3000/api-docs

## Installation

```bash
$ npm install
$ npm install -g @nestjs/cli
```

## Running the app

- dev mode
  ```bash
  $ npm run start:dev
  ```
- 컴파일된 js 파일 실행
  ```bash
  $ npm run start:prod
  ```

## Test Coverage

- coverage 확인

```bash
$ npm run test:cov
```

- unit test 확인

```bash
$ npm run test
```
